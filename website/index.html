Get Started

Become a validator in one line

Linux / macOS

curl -sSL https://spirachain.org/install.sh | bash

Windows

iwr -useb https://spirachain.org/install.ps1 | iex

Manual Installation

1

Clone Repository

git clone https://github.com/iyotee/SpiraChain.git
2

Build

cargo build --release
3

Create Wallet

./target/release/spira wallet new --output wallet.json
4

Start Validator

./target/release/spira node --validator --wallet wallet.json
